# Pasta para dashboards do Grafana

Coloque aqui arquivos `.json` exportados do Grafana para carregamento automático.

## Como exportar um dashboard:
1. Abra o dashboard no Grafana
2. Clique no ícone de compartilhamento (Share)
3. Vá em "Export" → "Save to file"
4. Salve o .json nesta pasta
5. O Grafana recarrega automaticamente em até 30 segundos
